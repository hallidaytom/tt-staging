How it works

The website is built as a standalone project — quoted, built, and delivered. Once it's live, the client moves onto a monthly plan that keeps everything running: hosting, email, domain, security, backups, and ongoing support. Any major new features or development down the track gets scoped and quoted separately.

What's included
🧱 Core Infrastructure

Domain, DNS, website hosting. All the behind-the-scenes stuff that keeps the site live. The client never has to touch it.

🧩 Systems

Website, business email, and enquiry forms. Set up and managed. Premium clients get e-commerce — taking payments online, product listings, and the compliance side that comes with it.

🛠️ Services

Content updates, fixes, and technical support. Something breaks, we deal with it. Menu needs updating, we update it. Major new development is scoped and quoted separately.

🛡️ Reliability

Daily backups, basic security hardening, and hosting that doesn't fall over. Monthly status report included for all clients.

🚀 Setup & Migration

Switching from another provider? We handle the full transition — domain, DNS, email, content, the lot. Test sites and staging environments available on request.

🔌 Integrations

Booking systems, payment gateways, ordering platforms — wired in properly, not just a link on a page. Standard platforms (Stripe, Shopify, Square) set up as part of the plan. Complex multi-system integrations scoped separately.

📡 Monitoring Premium

Active uptime checks and system monitoring. Problems get caught before the client notices.

🖥️ Application Hosting Premium

Custom web apps, booking platforms, internal tools. Anything that needs its own environment beyond a standard site.
